Politique de confidentialité – Vecto Livreur

Dernière mise à jour : 20 juin 2026

Cette politique de confidentialité décrit comment Mohamed Cheikh Ahmed (« nous », « notre ») collecte, utilise et protège les informations personnelles des utilisateurs de l'application mobile Vecto Livreur.

1. Données collectées

Lors de l'utilisation de l'application, nous collectons les données suivantes :

  • Numéro de téléphone : utilisé pour la création de compte et l'authentification via code OTP.
  • Nom complet : utilisé pour identifier le livreur sur la plateforme.
  • Localisation GPS : collectée en temps réel pendant les livraisons actives pour permettre au client de suivre sa commande.
  • Messages audio : enregistrés et envoyés uniquement lorsque le livreur choisit d'envoyer un message vocal dans le chat.
  • Photos : sélectionnées depuis la galerie et envoyées uniquement lorsque le livreur choisit d'envoyer une photo dans le chat.
  • Token de notification : utilisé pour envoyer des alertes de nouvelles commandes.

2. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Authentifier et identifier le livreur sur la plateforme Vecto.
  • Partager la position du livreur avec le client pendant la livraison.
  • Permettre la communication entre le livreur et le client via le chat intégré.
  • Envoyer des notifications push pour les nouvelles commandes disponibles.

3. Partage des données

Nous ne vendons, ne louons et ne partageons pas vos données personnelles avec des tiers, sauf dans les cas suivants :

  • La localisation GPS est partagée avec le client qui a passé la commande, uniquement pendant la livraison active.
  • Les messages (audio et photos) sont visibles par le client dans le chat de la commande concernée.

4. Stockage et sécurité

  • Les tokens d'authentification sont stockés de manière sécurisée sur l'appareil via expo-secure-store (chiffré).
  • Les communications entre l'application et le serveur sont effectuées via des connexions sécurisées (HTTPS / WSS).
  • Nous appliquons des mesures techniques raisonnables pour protéger vos données contre tout accès non autorisé.

5. Conservation des données

Vos données personnelles sont conservées tant que votre compte est actif. Lors de la déconnexion, les tokens locaux sont supprimés de l'appareil. Pour demander la suppression définitive de votre compte et de vos données, contactez-nous à l'adresse indiquée ci-dessous.

6. Permissions requises

  • Localisation : pour partager votre position avec le client pendant la livraison.
  • Microphone : pour envoyer des messages audio dans le chat.
  • Galerie photo : pour envoyer des photos dans le chat.
  • Notifications : pour recevoir les alertes de nouvelles commandes.

7. Droits des utilisateurs

Vous avez le droit de :

  • Accéder à vos données personnelles.
  • Demander la correction de données inexactes.
  • Demander la suppression de votre compte et de vos données.

Pour exercer ces droits, contactez-nous par e-mail.
